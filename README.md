# simple

## Project setup

### Clone repository
```
git clone https://github.com/Hatead1/simple-rete-vue-app
```
### Change directory
```
cd simple-rete-vue-app/
```
### Install npm modules
```
npm i
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
